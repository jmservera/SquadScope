- Meta descriptions should summarize the page value proposition in 155 characters or fewer.
- Structured data follows schema.org conventions and is validated before merge.
- Every content-generating script change gets an SEO impact check: will the output still rank?
- Prefer deterministic, content-derived titles over LLM-generated ones for reproducibility.
